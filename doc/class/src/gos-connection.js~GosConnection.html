    GosConnection

    GosConnections represent the connection to the GoshawkDB.

    They should be acquired through the static Goshawk.connect function.

    public transact(fn: function(txn: Transaction): {* | Promise<*, Error>}}): Promise<*, Error> source

    Queues a transaction for running, then ensures that transaction processing is happening. The returned promise resolves with the value returned by the fn once the transaction has committed. The fn may be run multiple times and so should avoid side effects. fn may be an asynchronous or a synchronous function.

    Params:

    NameTypeAttributeDescription
    fn function(txn: Transaction): {* | Promise<*, Error>}}

    the transaction function. This function may be run multiple times and should rethrow any TransactionRetryNeeded exceptions.

    Return:

    Promise<*, Error>

    a promise that resolves to the result of the transaction function once the transaction submits or an error if it cannot.
